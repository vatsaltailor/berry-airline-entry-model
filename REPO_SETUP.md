# Repository Description Setup

## GitHub Repository Description

To add a description that appears when you pin this repository:

1. Go to the repository on GitHub: https://github.com/vatsaltailor/berry-airline-entry-model
2. Click the **⚙️ (gear icon)** next to "About" on the right side
3. In the "Description" field, paste the following:

```
Python implementation of the Berry (1992) static entry game for U.S. airline markets. Estimates how market characteristics and competition affect airline entry decisions using maximum likelihood estimation.
```

4. Optionally add topics like: `econometrics`, `python`, `airline-industry`, `entry-models`, `maximum-likelihood`
5. Click "Save changes"

This description will now appear:
- When you pin the repository on your profile
- In the repository header below the name
- In search results

The description is also available in the `DESCRIPTION.txt` file in this repository for easy reference.
